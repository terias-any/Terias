# Terias

**Terias** là website cá nhân / portfolio của mình, được thiết kế theo phong cách hiện đại, tối giản, có hiệu ứng mượt và hỗ trợ cả trang giới thiệu riêng, trang lỗi tùy chỉnh, cùng giao diện sáng/tối tự động theo hệ thống.

## ✨ Tính năng nổi bật

* Giao diện hiện đại, bo tròn, glassmorphism nhẹ
* Hiệu ứng chữ xuất hiện dần và cuộn mượt
* Hiệu ứng gõ chữ ở phần lời chào
* Sidebar menu dạng nổi
* Tự động đổi giao diện sáng/tối theo hệ thống
* Trang `About me` riêng
* Bộ trang lỗi tùy chỉnh:

  * `400`
  * `403`
  * `404`
  * `500`
* Tối ưu cho mobile

## 🛠️ Công nghệ sử dụng

* **HTML5**
* **CSS3**
* **JavaScript**
* **Google Fonts**
* **Font Awesome**
* **Netlify** để deploy

## 🚀 Cách chạy

### Cách 1: mở trực tiếp

Chỉ cần mở file `index.html` bằng trình duyệt.

### Cách 2: chạy trên hosting

Upload toàn bộ thư mục lên Netlify, GitHub Pages, hoặc hosting khác.

## 🔗 Các trang chính

* Trang chủ: `index.html`
* Giới thiệu: `about-me/index.html`
* Trang lỗi: trong thư mục `errors/`
